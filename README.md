# student-management-system(c++)

A console-based menu-driven Student Management System built using C++.

## Features
- Add Student
- View Students
- Search Student
- Delete Student

## Technologies Used
- C++
- Object-Oriented Programming (OOP)
- STL (vector)

## How to Run
1. Compile:
   g++ main.cpp -o project
2. Run:
   .\project
